PROJECTS

  • HIDS & HIPS

    This project aims to secure host systems by implementing a Host Intrusion Detection and Prevention System.

    1. Real-time Monitoring: Tracks logins, file changes, and process behavior.
    2. Intrusion Detection: Identifies suspicious or malicious activity.
    3. Prevention Actions: Blocks harmful processes and unauthorized changes.
    4. Alert & Logs: Generates notifications and maintains records for analysis.

    Used Trend Micro Deep Security Agent as the main tool, this project strengthens host-level security and provides exposure to cybersecurity concepts.

  • UGV- Unmanned Ground Vehicle

    This project focuses on developing an Unmanned Ground Vehicle (UGV) integrating both hardware and software components for surveillance purposes. The system is designed to operate remotely in risky or inaccessible areas, providing live monitoring and enhanced safety. It uses wireless communication, camera-based vision, and autonomous navigation support for efficient movement. Key features include:

    1. Remote control and live video feed
    2. Obstacle detection and avoidance
    3. Real-time data transmission
    4.Useful for defense, disaster management, and industrial monitoring

    This project demonstrates practical application of robotics, IoT, and embedded systems in real-world scenarios.

  • FPS Video Game

    This project is a First Person Shooting (FPS) game created using Blueprint technology in Unreal Engine. The game allows players to experience immersive 3D environments with smooth controls and realistic shooting mechanics. Key features include:

    1. Developed entirely with Unreal Engine’s visual scripting (Blueprints).
    2. Interactive gameplay with player movement, aiming, and firing.
    3. Simple yet effective level design to demonstrate FPS mechanics.
    4. A strong foundation for further development into advanced AI enemies, multiple weapons, and scoring systems.

    This project highlights practical knowledge of game development fundamentals.
